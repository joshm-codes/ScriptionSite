Scription — jQuery Implementation:


This folder contains the jQuery-based version of the Scription project. It mirrors the functionality of the vanilla JavaScript version, but rebuilt using jQuery to practice DOM traversal, event handling, animations, and simplified selectors.

What This Version Covers:
- Rewriting core interactions using jQuery methods
- Using jQuery’s event system (.on(), .click(), etc.)
- Applying jQuery animations.
- Simplifying DOM manipulation compared to the JS    version
- Demonstrating equivalent functionality with a different approach

Purpose:
This version exists to show understanding of both vanillaJS and Jquery.  The ability to translate logic between frameworks.  Growth in front-end development patterns.
