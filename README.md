Project Overview:
A responsive landing page built as part of a course, expanded with custom Javascript features.

Features:
- Smooth scrolling
- Scroll-spy navigation
- Custom image slider with infinite looping
- Pagination controls
- CTA animation
- Auto-slide with hover pause
- Blockquote rotator with flip animation
- Tabbed content sections

Tech:
-HTML
-CSS
-Vanilla Javascript

Notes:
Some base HTML and CSS was provided by a course, but all interactive Javascript features were built by me.
